ML2SL gives contest operators the capability to use both programs together!
ML2SL is a one-way bridge to send essential qso info during contests.
MacLoggerDX runs as the main program for log entry. Whenever a QSO is logged,
essential information (time, callsign, mode and band) is also sent to SkookumLogger
so that its unique progress and tracking features can be used during contests.
